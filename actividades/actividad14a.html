De forma predeterminada, un cuadro tiene el tamaño lo suficientemente grande para mantener su contenido. Para establecer las dimensiones de un cuadro, se puede utlizar las propiedades: height y widht. Las formas más populares de especificar el tamaño de un cuadro son usar píxeles, porcentajes o EMs. Tradicionalmente, los píxeles han sido el método más popular, ya que permiten controlar con precisión su tamaño. Es este ejemplo se puede ver que se utiliza un elemento <div> que contiene 400 píxeles de ancho por 400 píxeles de alto.
