La etiqueta video: Para especificar la ubicación del archivo que se reproducirá, puede usar el elemento <source> dentro del elemento <video>. (Esto debe reemplazar el atributo SRC en la etiqueta <video> de apertura) tambien puede utilizar varios elementos <source> para especificar que el video está disponible en diferentes formatos. (Debido al error en el iPad, debe proporcionar el vídeo MP4 como el primer formato. De lo contratio, puede que no inicie.)

src: este atributo especifica la ruta de acceso al vídeo.

Type: se debe utilizar este atributo para decirle al navegador qué formato es el video. De lo contrario, descargará parte del vídeo para ver si puede reproducir el archivo (que tomará tiempo y ancho de banda).

Codecs: el códec que se usó para codificar el vídeo se suministra dentro del atributo Type. Tenga en cuenta el uso de comillas simples, así como comillas dobles en el atributo de tipo, cuando se suministra.
