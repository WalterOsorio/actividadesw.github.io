En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproducción automática y atributos de bucle utilizado en el ejemplo <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente, activa esa opción. Si se omite el atributo, la opción está desactivada. Si el navegador no admite el elemento <video> o el formato de vídeo utilizado, se mostrará lo que esté entre la <video> de apertura y las etiquetas de </video> de cierre.

Preload: este atributo le indica al navegador qué hacer cuando se carga la página. Puede tener uno de tres valores:
None: el navegador no debe cargar el vídeo hasta que el usuario presiona Play.
Auto: el navegador deve descargar el vídeo cuando se carga la páginaz.
Metadata: el navegador debe simplemente recopilas información como el tamaño, el primer fotograma, la lista de pistas y la duración.
