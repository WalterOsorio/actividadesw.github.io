<audio> HTML5 introdujo el elemento <audio> para concluir archivos de audio en sus páginas. Al igual que con el vídeo HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un número de atributos que le permiten controlar la reproducción del audio:

SRC: este atributo especifica la ruta de acceso al archivo de audio.

Controls: este atributo indica si el reproductor debe mostrar los controrles. Si no utiliza este atributo, no se mostrarán controles de forma preterminada. También puede especificar sus propios controles mediante JavaScript.

Autoplay: la presencia de este atributo indica que el audio debe empezar a reproducirse automáticamente. (Se considera una mejor práctica permitir que los visitantes elijan reproducir el audio).

Loop: este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado.
