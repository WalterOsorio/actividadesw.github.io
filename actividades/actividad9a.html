Cada elemento HTML puede llevar el aributo id. Se utiliza para identificar de forma única ese elemento de otros elementos de la página. Su valor debe comenzar con un letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no hay dos elementos en la misma página que tienen el mismo valor para sus atributos ID (de lo contrario, el valor ya no es único)

dar a un elemnto una identidad única le permite darle un estile diferente que cualquier otra instacia del mismo elemento de la página.

Por ejemplo, es posible que desee un párrafo dentro de la página (quizás un párrafo que contiene una cotización de extracción) un estilo diferente que todos los demás párrafos. En el ejemplo, el párrafo con el atributo ID cuyo es pullquote se hace en mayúsculas usando CSS.
