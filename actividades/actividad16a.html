Conceptos Clave en el Posicionamiento de Elementos

Construcción de Bloques

CSS trata cada elemento HTML como si estuviera en su propio cuadro. Este cuadro block-level comienzan en una nueva línea y actúan como las principales bloques de construcción de cualquier diseño, mientras que los inline-box fluyen entre el texto circundante. Puede controlar la cantidad de espacio que ocupa cada caja estableciendo el ancho de los cuadro (y aveces la altura también). Para separar los cuadros, puede usar bordes, márgenes, relleno y colores de fondo.

Normal Flow

Cada elemento block-level se encuentra en la parte superior de la siguiente. Puesto que esta es la forma predeterminada en que los exploradores tratan los elementos HTML, no se necesita una propiedad CSS para indicar que los elementos deben aparecer en el flujo normal.

La Evolución de la Bicicleta

En 1817, el Barón Karl Freiherr von Drais Inventó una máquina para caminar que le ayudaría a desplazarse más rápido por los jardines reales: dos ruedas en línea del mismo tamaño y una cabrilla en la parte delantera, acoplada en un marco donde él se montaba.
